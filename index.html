📌 프로젝트 비전

마인드펫은 단순한 헬스케어 앱이 아니라, 사용자의 감정과 신체 데이터를 비추는 거울입니다. 우리는 “지루한 관리”를 “돌봄과 성장”의 즐거움으로 바꿉니다.

🚩 우리가 해결하려는 문제

문제 1: 꾸준한 사용 어려움

  • 성과 지연 → 동기부여 상실
  • 목표 실패 → 죄책감·지루함 → 앱 회피
  • 망각, 피로, 시간 부족
  • 콘텐츠 단조로움, UX 불편, 커뮤니티 부재

문제 2: 실제 상태 반영 실패

  • 수동 입력 불편, 자동 수집 데이터 제한
  • 획일적 기준 적용, 맥락 반영 부족
  • 비적응형 솔루션 → 사후 리포트만 제공

💡 우리의 해결책

마인드펫은 펫 인터페이스 + 게이미피케이션 + AI 개인화를 융합합니다. 펫은 사용자의 신체적·정신적 데이터를 실시간 반영하며, 건강 관리를 돌봄과 성장의 즐거움으로 바꿉니다.
